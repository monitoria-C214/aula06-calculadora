## Como executar o projeto

```bash

# Clone este repositório
$ git clone https://github.com/monitoria-C214/calculadora-nodejs.git

# Acesse a pasta do projeto no seu terminal/cmd
$ cd calculadora-nodejs

# Instale as dependências
$ yarn install

# Execute os testes
$ yarn test

```
